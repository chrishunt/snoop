# Snoop `wip`
Monitor content for changes and be notified

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'snoop'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install snoop
```
